# Cybertron
Cybertron is a team in the [Digitopia](https://digitopia.gov.eg/) competition.

This project is an Arabic game that aims to teach users how to avoid malicious emails by simulating it as a type of training.

You can try the game now using the [project link](https://cybertron-digitopia.web.app/).
